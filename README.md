# Todo App

A todo_app made with Flutter

## Getting Started

This project is a starting point for a Flutter application.

## Installation

```bash
$ flutter pub get
```

## Running the app

```bash
# development
$ flutter run --debug
```

```bash
# production mode
$ flutter build apk
```
